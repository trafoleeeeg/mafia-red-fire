# MAFIA TRAFA — Gambling & Crypto трафик

Агентство performance-маркетинга для gambling и крипто проектов.

## Технологии

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Three.js

## Установка и запуск

```sh
# Клонировать репозиторий
git clone <YOUR_GIT_URL>

# Перейти в папку проекта
cd <YOUR_PROJECT_NAME>

# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev
```

## Сборка для продакшена

```sh
npm run build
```
